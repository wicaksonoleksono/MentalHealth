Prompt Analisis

Format yang digunakan secara otomatis:

{
  "indikator_1": {
    "penjelasan": "penjelasan untuk indikator 1",
    "skor": 0
  },
  "indikator_2": {
    "penjelasan": "penjelasan untuk indikator 2", 
    "skor": 0
  },
  ...
}

Skor: 0=Tidak Ada, 1=Ringan, 2=Sedang, 3=Kuat

Format ini di-hardcode untuk konsistensi parsing dan mendukung perubahan indikator.
